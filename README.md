Purpose:

I have had practically no experience with SQL up till now and thought this project would be a great way to get started.

Description:

This program creates a to-do list database, which will then prompt the user with 5 options: Add Task, View Tasks, Update Task, Delete Task, and Exit.  When adding a task it will be assigned a unique ID, name, description, date added, and days until due.
